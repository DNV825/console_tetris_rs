# console_tetris_rs

Rustの勉強。

- やりたい
  - [x] キー入力を判定する。
  - [ ] ブロックを落下させる。
  - [ ] ブロックを左右に動かす。
    - 左右キーで動かす。
  - [ ] ブロックを回転させる。
    - スペースキーで回転させる。
  - [ ] ブロックを消す。
    - 下の方から接触を判定し、接触していた場合はその場に固定する。
    - 接触判定後、消せるかを判定する。
    - 固着した行から上に4行チェックする
    - すべてそろっていた場合は消し、上の行を下に移動させる。
    - 消せるものをすべて消す→移動させるの順で画面を更新する。
    - そのあと、次のブロックを下ろす。
  - [x] ESCキーで終了する。
- 後回し
  - [ ] Pで一時停止
  - [ ] "press [space] key to start!"を表示させる。
  - [ ] 点数計算する。
  - [ ] 次のブロックを表示させる。
